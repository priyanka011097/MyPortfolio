# Priyanka Shahasane — Portfolio

Personal portfolio of **Priyanka Shahasane** (Full Stack Engineer / AI Solution Architect), built with React + Vite.

## Stack
- **React 18** + **Vite**
- **React Router** — `/` (Home), `/works` (Experience), `/gallery` (Projects)
- **Framer Motion** — page transitions & scroll-reveal animations
- **React Three Fiber + drei** — the interactive 3D "holo" crystal that travels down the page as you scroll (drag it to rotate)
- Vanilla CSS with custom properties for theming (dark / light)

## Features
- Fixed sidebar nav (HOME / EXPERIENCE / PROJECTS) with social links (LinkedIn, GitHub, Email, WhatsApp)
- Dark / light theme toggle (moon 🌙 / sun ☀️), persisted to `localStorage`
- Home: hero, About, Motivation, Skills (Frontend / Backend / AI & Cloud)
- Experience: stacked work history
- Projects: live homepage previews of shipped work
- Interactive scroll-driven 3D element, responsive layout, frosted skill chips

## Run locally
```bash
npm install
npm run dev      # http://localhost:5173 (or next free port)
npm run build    # production build to /dist
npm run preview  # preview the production build
```
